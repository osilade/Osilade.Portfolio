Sex & Marriage Sustainability
in Alimosho LGA

Qualitative Research In-depth Interviews Thematic Analysis Sociology

How education shapes intimacy — and why it matters for marriage

A qualitative sociology study exploring how sexual intimacy influences marriage sustainability among couples in Alimosho LGA, Lagos. Conducted through semi-structured in-depth interviews using purposive sampling, with thematic analysis applied to the findings.

The central finding: education plays a significant role in sexual compatibility and marital stability. Educated partners communicate more openly about intimacy, generating greater satisfaction and stronger emotional bonds.

Key finding: Women were found to be strongly influenced by emotional and intellectual connection. Uneducated or uncommunicative partners were consistently cited as a barrier to marital happiness — confirming that communication, not just attraction, sustains marriage.

05 — SQL

SQL Queries I Write

Real queries I run on Nigerian insurance data. Written in SQLite — straightforward, readable, and something I can explain line by line.

Beginner
Count Claims by Region
How many claims came from each region? The most basic question in any claims dataset.
SELECT region,
  COUNT(*) AS total_claims
FROM claims
GROUP BY region
ORDER BY total_claims DESC;
Beginner
Filter Settled Claims Only
Pull only successfully settled claims — the foundation of any settlement analysis.
SELECT claim_id, region,
  product, claim_amount
FROM claims
WHERE status = 'Settled'
ORDER BY claim_amount DESC;
Beginner
Average Claim by Product
Which product line has the highest average payout? Essential for pricing decisions.
SELECT product,
  ROUND(AVG(claim_amount),0)
    AS avg_claim_ngn
FROM claims
WHERE status = 'Settled'
GROUP BY product;
Intermediate
Settlement Rate by Region
What percentage of claims in each region were settled? COUNT combined with a percentage.
SELECT region,
  COUNT(*) AS total,
  SUM(CASE WHEN
    status='Settled'
    THEN 1 ELSE 0 END)
    AS settled
FROM claims GROUP BY region;
Intermediate
Top 10 Highest Claims
Find the 10 most expensive settled claims. Always the first thing a manager asks for.
SELECT claim_id, region,
  product, claim_amount
FROM claims
WHERE status = 'Settled'
ORDER BY claim_amount DESC
LIMIT 10;
Intermediate
Monthly Claims Trend
How many claims were filed each month? Reveals seasonality patterns across the year.
SELECT month,
  COUNT(*) AS claims,
  SUM(claim_amount)
    AS total_exposure
FROM claims
GROUP BY month
ORDER BY month;
